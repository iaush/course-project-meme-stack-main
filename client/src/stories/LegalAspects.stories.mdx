# Legal Aspects

All of the packages imported into our project are confirmed to be based on permissive opensource packages (specifically not copyleft/protective licenses such as GPL, AGPL).
As the application intends to be for knowledge and learning, we feel that an application that can be adapted and changed according to use case would be the most 
beneficial to help students. Should time allow, we feel that having the repository maintain a permissive opensource such as the MIT license would be best to allow it to be built upon 
without restrictive copy-left constraints should it be helpful.
